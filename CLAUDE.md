## 기능
- 모든 text 파일을 열수 있는 에디터
- .txt, .c, .py, .java, .md 등 텍스트 파일 형식의 문서를 열 수 있음. 편집 가능
- ms-word, hwp, pdf 문서도 열수 있음. 편집 불가
- 편집 가능한 문서는 기본 에디팅 기능이 모두 지원해야 함.
- 파일 열기, 저장 등 파일 관련 기능 지원
- 여러 페이지를 열 수 있으며, 탭으로 구분함.
- 줄 번호가 있어야 함.(예:  1: text .....), 줄번호는 전체 줄 수를 판단해서 텍스트의 시작이 왼쪽 정렬되도록 자동 들여쓰기가 되어야 함
- 텍스트 문구를 선택 복사 가능해야 함
- 폰트 변깅이 가능해야함
- 글자색, 바탕색을 선택이 가능해야 함.
- 기본 모드, 다크모드, 라이트 모드 지원
